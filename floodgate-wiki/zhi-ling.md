# 指令

|        指令       |                 权限                |                描述                |
| :-------------: | :-------------------------------: | :------------------------------: |
|   `fwhitelist`  |   `floodgate.command.whitelist`   | 给 Floodgate 设立白名单。点击 这里=== 查看详情。 |
|  `linkaccount`  |  `floodgate.command.linkaccount`  |       将当前账号绑定到指定 Java 版账号。       |
| `unlinkaccount` | `floodgate.command.unlinkaccount` |               取消绑定。              |
